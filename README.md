# Aman Cajoler - Ethiopian Marketplace

Buy and sell houses and cars across Ethiopia.

## Features
- Houses and Cars
- 9 Ethiopian regions
- 7 Languages
- Admin panel

## Quick Start
1. Backend: `cd backend && npm install && npm start`
2. Frontend: `cd frontend && npm install && npm run dev`
3. Visit: http://localhost:3000
